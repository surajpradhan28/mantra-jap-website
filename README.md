# Mantra Jap Website

This is the official website for **Mantra Jap: Chanting Counter** app.

## About the App

Mantra Jap is a comprehensive spiritual companion app that helps users practice mantra meditation, perform daily aarti, and maintain their spiritual journey with features like:

- Digital chanting counter with beautiful mala beads
- Extensive mantra library in multiple languages
- Daily aarti collection with synchronized lyrics
- Hindu festival calendar
- Breathing & meditation exercises
- Digital puja room

## Download
